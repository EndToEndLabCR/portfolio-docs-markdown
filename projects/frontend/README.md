# Portfolio App - Frontend Project

## Overview

This is the frontend application for the Portfolio App, built with React, TypeScript, and Ant Design.

---

## Tech Stack

- **Framework**: React 18+
- **Language**: TypeScript 5.0+
- **Build Tool**: Vite
- **UI Library**: Ant Design 5.10+
- **State Management**: Redux Toolkit
- **Routing**: React Router 6+
- **Styling**: CSS/SCSS
- **Testing**: Vitest + React Testing Library
- **HTTP Client**: Axios

---

## Prerequisites

- Node.js 18+
- npm 9+ or yarn
- Git
